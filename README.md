## لماذا اخترنا هذا النشاط
محل ورد في الدمام بتقييم مرتفع (4.9) وصور حقيقية جذّابة (باقات + واجهة + داخل المحل)، نشاط بصري بطبيعته يستفيد كثيراً من موقع أنيق يعرض الأعمال ويسهّل الطلب عبر واتساب.

## فكرة التصميم
"بوتانيكال عصري" — لوحة ألوان من الأخضر الميرمية (sage) والوردي الخفيف (blush) والكريمي الدافئ مع نص أخضر غامق. خط العناوين **El Messiri** وخط النص **Tajawal**. حركات هادئة تليق بالورد: بتلات منسابة في الهيرو، ظهور الكروت بتأثير "تفتّح" عند التمرير، تكبير بطيء (ken-burns) لصورة الهيرو، تكبير الصور عند المرور، ولمعة خفيفة على زر الطلب.

## التقنية
HTML/CSS/JS عادي بدون أي build step ولا مكتبات ثقيلة ولا `node_modules` / `package.json` في الموقع. خطوط قوقل عبر preconnect + `display=swap`. جاهز لـ GitHub Pages (يحتوي `.nojekyll`).

## حالة الـ backend
لا يوجد نظام حجز فعلي — الموقع نسخة تعريفية. نموذج الطلب:
1. يتحقق من المدخلات (الاسم، الجوال، نوع الطلب).
2. يحفظ نسخة تجريبية في `localStorage` (المفتاح `fs_orders`).
3. يفتح **واتساب** برسالة مُعبّأة مسبقاً بتفاصيل الطلب، ويعرض إشعار نجاح (toast).

## مصدر الصور
صور حقيقية من ملف النشاط على خرائط قوقل (باقات + داخل المحل + واجهة المحل بلافتته الخاصة). تمت مراجعة كل صورة واستبعاد ما يحمل لافتات منافسين أو نشاطاً خاطئاً أو وجوهاً واضحة. جميع الصور المعروضة تخص Flower Stalk.
